# Data_Visualization_proj1
ETL Pipeline project on a Global Disaster Dataset 

##  Overview

This project implements an ETL (Extract, Transform, Load) pipeline on a global disaster dataset. The data was processed, cleaned, and analyzed using Python and MySQL.

---

##  Dataset

The dataset used is the **EM-DAT Global Disaster Dataset** from Kaggle. It contains worldwide disaster records including disaster type, country, dates, deaths, affected population, and economic damage.

---

##  ETL Steps

* Extracted data using Pandas
* Cleaned and handled missing values
* Split into two tables: Disaster Events & Disaster Impact
* Loaded into MySQL
* Performed SQL JOIN queries for analysis

---

##  Tools Used

Python, Pandas, MySQL, Jupyter Notebook, GitHub
